# Тестовый README


## Каждый последующий заголовок будет меньше предыдущего

В этом месте проверка разрыва строки. Один текст находится сдесь,<br>а другой сдесь


### Текст курсивом

Можно пометить за счет * по краям слова, что бы вышло вот *так*

Или можно использовать _ и получится вот _так_

----

#### Полужирный текст

Для этого нужны уже 2 звезды (**), **вот результат**

Или 2 нижних подчеркиваня (__), __пример как это будет выглядеть__

----

##### Зачеркнутый текст

А тут 2 волнистые линии (~~), ~~зачеркнутый текст~~



###### Ненумерованные списки

Нужно вначале поставить либо *, либо -

* Первая звездочка
* Вторая звездочка

- Первая черточка
- Вторая черточка

----

###### Нумерованные списки

Ничего удивительного, нужны цифры с точкой

1. Первый
2. Второй
